# Advanced Personal Finance & Investment Management System

A complete finance dashboard where users can manage income, expenses, savings goals, budgets, debts, and investments. All data is stored using LocalStorage in the browser.

## Features

- **Dashboard**: Overview with total income, expenses, savings, investments, net balance, and charts.
- **Income Manager**: Track income sources with amounts, dates, and categories.
- **Expense Tracker**: Log expenses by category and date.
- **Budget Planner**: Set monthly budgets and track spending.
- **Savings Goals**: Create goals and track progress with progress bars.
- **Debt Manager**: Manage debts and payments.
- **Investment Tracker**: Track investments and profit/loss.
- **Analytics**: Charts for expenses, income, savings, and investments.
- **Reports**: Monthly and yearly financial reports.
- **Settings**: Change currency, dark mode, export data, reset data.

## Technologies

- HTML5
- CSS3 with modern gradients, shadows, and animations
- JavaScript (ES6)
- Chart.js for visualizations
- LocalStorage for data persistence

## Getting Started

1. Open `index.html` in your web browser.
2. Navigate through the sidebar to different sections.
3. Start adding your financial data.
